# IssueFlow AI

Modern AI-powered issue tracking and collaborative project management platform built for developers.

## Features

- AI-assisted issue tracking
- Real-time collaboration
- Kanban boards
- Project discussions
- Wiki system
- Analytics dashboard
- GitHub-style activity feeds

## Stack

### Backend
- FastAPI
- PostgreSQL
- Redis
- WebSockets

### Frontend
- React
- TypeScript
- TailwindCSS

### DevOps
- Docker
- GitHub Actions
- Prometheus
- Grafana
